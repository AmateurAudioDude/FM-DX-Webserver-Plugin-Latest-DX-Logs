## Instructions

* [Download the latest zip file](https://github.com/AmateurAudioDude/FM-DX-Webserver-Plugin-Latest-DX-Logs/archive/refs/heads/main.zip)
* Extract `LatestDXLogs` folder, and `LatestDXLogs.js` to **FM-DX Webserver** `plugins` folder
* Restart **FM-DX Webserver** if required
* Login to **Adminstrator Panel** and enable plugin
* Restart **FM-DX Webserver** again if required
* Server-side configuration options stored in `/plugins_configs/LatestDXLogs.json`
* Client-side configuration options located in `pluginLatestDXLogs.js`
